# Программа принимает на вход строку - разделитель, вам необходимо распечатать числа от 1 до 5, выводя между ними введенный разделитель

В этой задаче мы сами будем решать, какое значение использовать в качестве разделителя в параметре sep.

Программа принимает на вход строку - разделитель, вам необходимо распечатать числа от 1 до 5, выводя между ними введенный разделитель[^1].

```
Sample Input 1:
!
Sample Output 1:
1!2!3!4!5

Sample Input 2:
$
Sample Output 2:
1$2$3$4$5

Sample Input 3:
BoB
Sample Output 3:
1BoB2BoB3BoB4BoB5
```

## IDLE log

```python
# Рассуждение с четким примером ввода/вывода
# например, «Если ввожу !, должен получить A!2!3!4!5»,
# 'сущность1'sep'сущность2'sep'сущностьN'
one = 1
two = 2
three = 3
four = 4
five = 5
separator = input()
print(one, two, three, four, five, sep = separator)
>>> !
```
