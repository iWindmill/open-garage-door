# Подсчитать, сколько всего сульфида необходимо на обработку всех прямоугольных панелей с обеих сторон

Известно, что на обработку одного квадратного метра панели требуется 1г сульфида. Всего необходимо обработать N прямоугольных панелей размером A на B метров. Вам необходимо подсчитать, сколько всего сульфида необходимо на обработку всех панелей. `И не забудьте, что панели требуют обработки с обеих сторон.`

На вход программе подаются 3 положительных целых числа N,A,B _в одну строку_

PS: в этой задаче обратите внимание на формат ввода. Числа вводятся в одну строку, поэтому вы уже не можете их так считать:

```python
n = int(input())
a = int(input())
b = int(input())
```

Возникнет ошибка! Чтобы считать три значения, которые поступают через пробел в одной строке, пользуйтесь следующей инструкцией:

```python
n, a, b = map(int, input().split()) 
```

Обращайте внимание на формат ввода!

```
Sample Input 1:
5 2 3
Sample Output 1:
60

Sample Input 2:
14 23 5
Sample Output 2:
3220
```

## Какую информацию успешно использовал?
- S. Площадь прямоугольника - внутренняя часть плоской фигуры. Площадь принято обозначать латинской буквой S. измеряется площадь: в квадратных миллиметрах (мм²), квадратных сантиметрах (см²), квадратных дециметрах (дм²) и так далее.  Чтобы найти площадь прямоугольника, необходимо длину стороны умножить на ширину[^3]
