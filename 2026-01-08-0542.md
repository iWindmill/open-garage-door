# Инди-курс программирования на Python

## Упражнения

- Сумма двух чисел. Напишите программу, которая принимает на вход два целых числа в отдельных строках и выводит на экран их сумму[^1]
- Площадь и периметр прямоугольника. В этом задании необходимо написать программу, которая вычисляет площадь и периметр прямоугольника по введенной длине и ширине. Входные данные - на вход программе в отдельных строках поступают два вещественных числа a и b: длина и ширина прямоугольника. Выходные данные - программа должна вывести через пробел два значения: сперва площадь S, а затем периметр P прямоугольника[^1]
- Из Фаренгейтов в градусы Цельсия. Дано значение температуры в градусах Фаренгейта. Определить значение этой же температуры в градусах Цельсия. Входные данные - вещественное число F  - температура в градусах по Фаренгейту. Выходные данные - градусы Цельсия[^1]
- Найдите результат выражения `|a| + |b|`. Значения переменных `а` и `b` поступают на вход в отдельных строках и могут быть только целого типа[^1]
- Длина отрезка. Напишите программу, которая вычисляет длину отрезка (т.е. расстояние между двумя точками), заданного двумя значениями `x1` и `x2` (вещественные числа)[^1]

input(). Функция позволяет получить информацию, которую пользователь вводит с клавиатуры[^1]

P. Периметр - сумма длин сторон многоугольника. Периметр принято обозначать латинской буквой Р. Чтобы найти площадь прямоугольника, необходимо длину стороны умножить на ширину. Измеряется в в миллиметрах (мм), сантиметрах (см), дециметрах (дм) и других единицах измерения. Для нахождения периметра необходимо сложить длины всех сторон[^3]

S. Площадь прямоугольника - внутренняя часть плоской фигуры. Площадь принято обозначать латинской буквой S. измеряется площадь: в квадратных миллиметрах (мм²), квадратных сантиметрах (см²), квадратных дециметрах (дм²) и так далее.  Чтобы найти площадь прямоугольника, необходимо длину стороны умножить на ширину[^3]

Type casting. Преобразование типа данных - базовая операция практически в любом языке программирования, позволяющая выполнить преобразование из одного типа данных в другой[^1]
