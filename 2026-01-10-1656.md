```
Sample Input 1:
0.123456789
Sample Output 1:
0.12 0.123

Sample Input 2:
45.789437823721
Sample Output 2:
45.79 45.789
```

## Какие знания понадобились?

- IDLE. Интегрированная среда разработки и обучения Python[^2]
- input(). Функция получения информации, которую пользователь вводит с клавиатуры[^1]
- round(). `(number[, ndigits])`. Функция возврата числа, округленного до ndigits точности после десятичной точки. Если n-цифр опущено или равно None, возвращает ближайшее целое число к заданному значению[^3].
- type casting. Преобразование типа данных - базовой операцией практически в любом языке программирования, позволяющей выполнить преобразование из одного типа данных в другой[^1]
