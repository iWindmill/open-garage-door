# Принять на вход три целых числа в одной строке через пробел и выводит их последовательно через запятую

Напишите программу, которая  принимает на вход три целых числа в одной строке через пробел и выводит их последовательно через запятую как в примерах ниже[^1].

```
Sample Input 1:
n
Sample Output 1:
n

Sample Input 2:
n
Sample Output 2:
n

Sample Input n:
n
Sample Output n:
n
```

## IDLE log

```python
# Рассуждение с четким примером ввода/вывода
# например, «Если ввожу три целых числа в одной строке через пробел
# 1 2 3, то на выходе получаю из последовательность через запятую
# 1, 2, 3

one, two, three = map(int, input().split())
print(one, two, three)

>>> 1 2 3
# result manual running in IDLE: fail
```

```python
one, two, three = map(int, input().split())
print(one, two, three, sep = ',')

>>> 1,2,3
# result manual running in IDLE: done
```

## Какую информацию успешно использовал?
- end. Parameter of the print(). Default - `\n` [^1].
- print(). `print(value, ..., sep=' ', end='\n')`[^1].
- sep - «separator». print(value1, value2, ..., valueN, sep=new_separator)[^1].
	```python
	print(1, 2, 3)
	print(1, 2, 3, 4, sep=' ')
	print(1, 2, 3, 4, 5, sep='')
	print(1, 2, 3, 4, 5, 6, sep=',')
	print(1, 2, 3, 4, 5, 6, 7, sep='*')
	print(1, 2, 3, 4, 5, 6, 7, 8, sep='###')
	print(1, 2, 3, 4, 5, 6, 7)
	```
- TypeError: sep must be None or a string, not int. For case `print(1, 2, 3, 4, sep=123)`[^1].
