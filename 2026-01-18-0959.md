# Шаблон для оформления задачи

Сказала она! Напишите программу, которая: Принимает на вход строку (предложение); Выводит то же самое предложение, но в конце добавляет "Сказала она!" определенным образом (см. примеры). Пользуйтесь аргументом end  и следите за символами, которые печатаются.[^1].

```
Sample Input 1:
hello
Sample Output 1:
hello - Сказала она!

Sample Input 2:
easy peasy
Sample Output 2:
easy peasy - Сказала она!
```

## IDLE log

```python
# Рассуждение с четким примером ввода/вывода
# например, «Если ввожу 'hello', должен получить
# 'hello - Сказала она!'»,
phrase = input()
print(phrase, end = ' - Сказала она!')
# result manual running in IDLE: success
# result running in Stepik: done
```
