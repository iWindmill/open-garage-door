```python
Sample Input 1:
45 80 98 47 81
Sample Output 1:
98

Sample Input 2:
1 86 63 98 20
Sample Output 2:
98
```

## Как улучшить качество выполняемой работы?
- Следует отказаться от копирования и вставки, так только оказываю себе медвежью услугу.
- Следует взять за правило использовать рассуждение с доказательствами - приводить сначала четкий пример ввода/вывода. Написать сначала: «Если ввожу 3 7 2 9 1, должен получить 9» — и только потом код.

## Что не понял?
- В чем суть `TypeError: 'int' object is not iterable`. Нужно будет глубже пройтись по:
	- `max(5)` → ошибка (5 — не итерируемый объект)
	- `max([5])` → работает (список — итерируемый)
	- `max(a, b, c)` → работает (несколько позиционных аргументов)
