# Два бандита. Сколько банов успеть прострелить один и не успел прострелить второй

[Два бандита](https://acmp.ru/asp/do/index.asp?main=task&id_course=1&id_section=1&id_topic=26&id_problem=146). Гарри и Ларри решили пострелять по банкам, выставленным на бревно. Они стреляли одновременно, но с разных сторон:

    ➖ Гарри стрелял слева направо;

    ➖ Ларри стрелял справа налево.

В какой-то момент их пули попали в одну и ту же последнюю банку, после чего стрельба прекратилась.

Теперь они спорят, кто кому должен деньги за то, что помешал насладиться стрельбой. Однако **они не помнят, сколько банок было изначально**, а лишь знают два значения `a` и `b`:
- `a` — сколько банок успел прострелить Гарри;
- `b` — сколько банок успел прострелить Ларри.

Ваша задача — определить:
1. Сколько банок **не успел прострелить Гарри;**  
2. Сколько банок **не успел прострелить Ларри**.

### **Входные данные:**

Программа получает два целых числа в одной строке:
- a (1 ≤ a ≤ 10) — количество банок, которые прострелил Гарри;
- b (1 ≤ b ≤ 10) — количество банок, которые прострелил Ларри.

### **Выходные данные:**
Выведите два числа через пробел:

- сперва количество банок, которые **не успел прострелить Гарри**.
- затем количество банок, которые **не успел прострелить Ларри**.[^1].

```
Sample Input 1:
4 7
Sample Output 1:
6 3

Sample Input 2:
5 3
Sample Output 2:
2 4

Sample Input 3:
2 7
Sample Output 3:
6 1
```

# Рассуждение с четким примером ввода/вывода

# Если Гарри прострелил слева направо 4 банки,
# А Ларри прострелил справа налево 7 банок
# то Гарри не усперил прострелить 6 банок
# а Ларри не успел прострелить 3 банки

# Как это выглядит визуализировал: 1845-1848
# Начинается так
# [G] [] [] [] [] [] [] [] [] [L]
# До того, как оба стрельнут в последнюю банку одновременно
# [G] [G] [G] [] [L] [L] [L] [L] [L] [L]
# Последняя банка
# [G] [G] [G] [GL] [L] [L] [L] [L] [L] [L]
# Осталось Garry - 6 банок, которые безусловно Larry взял
# Осталось Larry - 3 банки, которые безусловно взял Garry

# Соотношение нашел: 1848-1852
# Безусловные банки Гарри = количество простреленных - банка совместная
# Безусловные банки Гарри = 4 - 1 = 3
# Все что было безусловно у Гарри является потерей Ларри
# Безусловные банки Ларри = количество простреленных - банка совместная
# Безусловные банки Ларри = 7 - 1 = 6

# Coding: 1853-1856
harry, larry = map(int, input().split())
result = (larry - 1, harry - 1)
